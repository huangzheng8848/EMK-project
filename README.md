# Projektstruktur und Dateibeschreibung

- Manuelle_Hysteresemessung/
  Programm zur manuellen Messung des Hysterese-Effekts (Strom-Dehnung). Hinweis: Das Array muss für jede Messung manuell geändert werden.

- Messergebnisse + Invers_control/
  Beinhaltet die gesammelten Messdaten sowie das Hauptprogramm (Invers control.py)  zur Positionsregelung.

- Stromregler/
  Arduino-Programm zur Stromregelung der Hardware.

- Test_Ergebnisse/
  Ordner für die Testergebnisse.

- Visualisierung der Messergebnisse.py
  Python-Skript zur grafischen Darstellung (Plotten) der Ergebnisse aus der manuellen Hysteresemessung.
